# MySchedule
课程表应用
课程表应用是小生独立设计、开发的一款应用，至今因为时间紧迫，还未完成，只做了UI部分。
该应用暂时有四个Activity（待完善），主界面展示课程信息，添加课程表界面，管理课程表界面，设置界面。

主界面。因为还未实现功能，所以所用的数据都是假的。使用了ViewPager + TabLayout实现水平滑动界面，右边有周六周日tab未显示。
